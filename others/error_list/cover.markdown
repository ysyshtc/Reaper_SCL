常见错误 \begin{itemize}
	\item 题意不清
	\item 传达题意失误
	\item Clarifications更改了题意

	\item 题目中的常量打错
	\item Input, Output格式不对
	\item memset, sizeof的变量名不一致
	\item 构造函数写错
	\item 多个常量弄混
	\item 编译器选择不正确
	\item 改动队友代码
	\item 注释语句没删除干净

	\item double输出是\%f
	\item 使用类似\\0等奇怪的东西
	\item 逗号表达式返回值

	\item STL没有清空
	\item int, double弄混
	\item 变量含义与类型不一致
	\item STL爆界可以导致奇怪的错误

	\item sqrt, 输出取整的精度问题
	\item 实数比较的精度问题

	\item cin读入超时
	\item 过渡使用memset, memcpy导致的超时
	\item 空STL占用的空间也很大

	\item 新加入的代码与之前的不兼容
\end{itemize}
